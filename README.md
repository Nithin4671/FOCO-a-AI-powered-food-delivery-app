# FOCO - FOODCONNECT

FoodConnect is an AI-powered food donation management platform designed to reduce food wastage and support sustainability goals.

The system connects food donors with NGOs by enabling easy donation tracking, monitoring expiry dates, and providing AI-based insights for urgent food distribution.

---

## Features

- User authentication (Login & Register)
- Secure session-based access
- Add and manage food donations
- View available donations
- Dashboard with real-time statistics
- AI insights for urgent food items
- Clean and responsive UI
